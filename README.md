# PvZ3 Wwise Project

> Reverse engineering the PvZ3 Wwise project to recreate the original audio system.

---

## Overview

Reconstruction of the original *Plants vs Zombies 3* Wwise project:

- Audio events
- SoundBanks
- Hierarchies
- RTPCs / States / Switches

## How to Use

- Install **Wwise 2023.1.19** with **Android** as the deployment platform  
- Download it from [Audiokinetic Download Page](https://www.audiokinetic.com/ja/download)  
- You will need an Audiokinetic account to download the launcher
